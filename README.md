# RN-puppy-care

To install dependencies:

<pre>
npm install
</pre>

To run:

1.expo 설치

<pre>npm install -g expo-cli</pre>

2.Mac 유저면, watchman 설치

<pre>
brew update 
brew install watchman
</pre>

3.웹뷰로 보려고 하는 로컬 웹 App.tsx에 작성 (ex. url:"http://localhost:3000/")

4.expo 실행

<pre>
expo start
</pre>

5.XCode 설치 후 i 입력
